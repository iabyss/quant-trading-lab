# Fundamental Analysis Skill

基本面分析工具，提供财务数据、估值指标和选股筛选功能。

## 支持的功能

### 1. get_financial_data 获取财务数据

获取公司财务报表数据。

**参数:**
- `symbol`: 股票代码
- `statement_type`: 报表类型 (income/balance/cashflow)
- `period`: 周期 (annual/quarterly)

**返回:**
- 营收、利润、资产、负债
- 盈利能力指标
- 现金流数据

**使用示例:**
```
获取苹果的财务数据
查询茅台的利润表
显示腾讯的资产负债表
```

### 2. get_valuation 获取估值数据

获取估值指标。

**参数:**
- `symbol`: 股票代码
- `price`: 当前价格

**返回:**
- PE、PB、PS、PCF
- 市值
- 股息率
- PEG

**使用示例:**
```
查询茅台估值
获取苹果的PE和PB
显示特斯拉估值分析
```

### 3. get_financial_ratios 获取财务比率

获取关键财务比率。

**返回:**
- ROE、ROA
- 毛利率、净利率
- 资产负债率
- 流动比率

**使用示例:**
```
查询ROE指标
获取毛利率数据
显示财务比率分析
```

### 4. screen_stocks 选股筛选

基于基本面条件筛选股票。

**筛选条件:**
- PE范围
- ROE要求
- 净利润增长
- 股息率
- 资产负债率

**使用示例:**
```
筛选PE<20的股票
找ROE>15%的股票
选择高股息股票
```

### 5. compare_stocks 对比股票

对比多只股票的基本面。

**参数:**
- `symbols`: 股票代码列表

**对比内容:**
- 估值对比
- 盈利对比
- 成长性对比

**使用示例:**
```
对比茅台和五粮液
比较苹果和微软
```

### 6. get_industry_analysis 行业分析

分析所属行业状况。

**返回:**
- 行业景气度
- 竞争格局
- 上下游关系

**使用示例:**
```
分析白酒行业
查询新能源汽车行业
显示银行板块分析
```

## 常用财务指标参考

| 指标 | 说明 | 优秀值 |
|------|------|--------|
| ROE | 净资产收益率 | >15% |
| ROA | 总资产收益率 | >5% |
| 毛利率 | 毛利率 | >30% |
| 净利率 | 净利润率 | >10% |
| PE | 市盈率 | 10-25 |
| PB | 市净率 | 1-5 |
| 资产负债率 | 负债/资产 | <60% |
| 流动比率 | 流动资产/流动负债 | >1.5 |

## 使用流程

1. 用户查询财务数据 → 使用 `get_financial_data`
2. 用户查询估值 → 使用 `get_valuation`
3. 用户筛选股票 → 使用 `screen_stocks`
4. 用户对比股票 → 使用 `compare_stocks`
5. 用户分析行业 → 使用 `get_industry_analysis`

## 数据来源

- 使用项目内的 `data/fundamental.py` 模块
- 财务数据需要接入真实数据源 (Tushare/AKShare)
