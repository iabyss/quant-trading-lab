# 短期交易策略开发框架 - 讨论纪要

## 讨论主题
**高风险高收益短期交易策略开发**
目标资金：30万

---

## 参与成员
- 系统架构师
- 策略设计师
- 风控设计师
- 数据架构师

---

## 讨论过程

### 1. 策略定位讨论

**策略设计师**: "30万资金，目标高风险高收益，建议做短线交易。考虑几个方向：
1. 日内超短（但需要实时盯盘）
2. 3-5天短线波段
3. 龙头股追涨"

**风控设计师**: "高收益意味着高风险。30万资金量，建议：
- 单票止损10%
- 单日亏损5%熔断
- 总仓位不超80%"

**系统架构师**: "同意。策略框架需要：
- 快速信号生成
- 实时仓位监控
- 自动化风控执行"

---

### 2. 策略逻辑讨论

**策略设计师**: "我建议用MACD+成交量共振策略：
- MACD金叉买入
- 放量突破确认
- 止损5%，止盈15%
- 持有3-5天"

**数据架构师**: "A股数据有个问题：T+1限制。当天买的不能卖。
建议：
- 尾盘2:30买入
- 次日开盘卖出
- 避开盘中波动"

**风控设计师**: "同意T+1方案。另外建议：
- 早盘只观察，不买入
- 14:30后决策
- 收盘前5分钟执行"

---

### 3. 仓位管理讨论

**风控设计师**: "30万资金，建议：
- 单票最大30%仓位（9万）
- 最多同时持有2-3只
- 止损线：-5%减半仓，-10%清仓"

**策略设计师**: "再加一条：
- 盈利10%后，止损上移至成本价
- 让利润奔跑"

---

### 4. 选股池讨论

**策略设计师**: "高收益需要找弹性大的股票：
- 题材股、龙头股
- 创业板、科创板
- 流通盘50亿以下"

**数据架构师**: "建议建立动态股票池：
- 每日9:30更新涨停板
- 过滤ST、北交所
- 保留10只候选"

---

### 5. 执行流程设计

**系统架构师**: "综合讨论结果，执行流程：

```
9:30-14:30  观察候选池，等待信号
14:30       筛选符合MACD金叉+放量的股票
14:55       执行买入（尾盘）
次日9:30    检查持仓
次日14:30   决定卖出或持有
```

---

### 6. 风控规则汇总

**风控设计师**: "最终风控规则：

| 规则 | 参数 |
|------|------|
| 单票止损 | -5% |
| 单票止盈 | +15% |
| 单日熔断 | -3% |
| 单票上限 | 30% |
| 总仓上限 | 80% |
| 持仓上限 | 2只 |

---

## 决策结果

### 策略框架

| 项目 | 内容 |
|------|------|
| 策略名称 | 短线波段策略 |
| 交易周期 | 1-5天 |
| 选股条件 | MACD金叉+放量突破 |
| 买入时间 | 尾盘14:55 |
| 卖出时间 | 次日尾盘14:30 |
| 止损 | -5% |
| 止盈 | +15%或移动止损 |

### 资金管理

| 项目 | 参数 |
|------|------|
| 初始资金 | 30万 |
| 单票仓位 | 30%（9万） |
| 最大持仓 | 2只 |
| 总仓上限 | 80%（24万） |

### 选股池

- 创业板/科创板优先
- 流通盘<50亿
- 近期有涨停板
- 非ST/*ST
