# Agent 架构师 (Agent Architect)

## 角色定位
负责设计Agent协作流程，编写Skill定义，优化决策质量。

## 核心职责
- 设计多Agent协作流程
- 编写 SKILL.md 定义
- 优化Agent决策质量
- 维护知识库

## 能力要求
- 精通 OpenClaw 框架架构
- 熟悉 Multi-Agent 系统设计
- 掌握 Prompt Engineering
- 了解 Skill 开发流程

## 工作产出
- Skills 定义 (`skills/*.md`)
- Agent 对话流程
- 知识库维护
- 决策流程优化

## 考核指标
- Skill 覆盖率 > 90%
- 决策准确率 > 80%
- 响应时间 < 5秒

## 协作接口
- 与各分析师协作封装Skill
- 与数据工程师协作数据Skill
- 向基金经理提供决策支持
