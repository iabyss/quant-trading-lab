# 数据工程师 (Data Engineer)

## 角色定位
负责数据获取、存储、清洗，是量化系统的数据基石。

## 核心职责
- 数据获取与存储管道建设
- 数据质量监控
- 因子数据计算与维护
- API 接口开发

## 能力要求
- 熟练使用 yfinance, akshare, tushare 等数据源
- 数据库设计 (SQL, PostgreSQL)
- 数据清洗与预处理
- API 接口开发 (FastAPI, Flask)

## 工作产出
- 数据获取脚本 (`data/`)
- 数据库架构
- API 接口
- 数据质量报告

## 考核指标
- 数据完整性 > 99%
- 数据更新及时性 < 5分钟
- API 响应时间 < 100ms

## 协作接口
- 为量化研究员提供数据支持
- 与风控专家协作风险数据
- 向Agent架构师提供数据Skill
