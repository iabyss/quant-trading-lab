# 量化交易决策团队

## 团队架构

```
team/
├── fundamental/           # 基本面分析团队
│   └── 基本面分析师.md
├── short_term/            # 短期交易策略
│   └── 短期交易策略分析师.md
├── medium_term/          # 中期交易策略
│   └── 中期交易策略分析师.md
├── long_term/            # 长期交易策略
│   └── 长期交易策略分析师.md
├── discussion/           # 讨论团队
│   └── 讨论团队.md
├── fund_manager/         # 基金经理
│   └── 基金经理.md
└── trader/               # 操盘手
    └── 操盘手.md
```

## 决策流程

```
┌─────────────────┐
│   基本面分析    │ ◄── 估值、价值评估
└────────┬────────┘
         │
    ┌────┴────┬──────────┐
    ▼         ▼          ▼
┌───────┐ ┌───────┐ ┌────────┐
│短期策略│ │中期策略│ │长期策略│
└───┬───┘ └───┬───┘ └───┬───┘
    │         │         │
    └────────┬┴────────┘
             ▼
    ┌────────────────┐
    │    讨论团队    │ ◄── 整合各方观点
    └────────┬───────┘
             ▼
    ┌────────────────┐
    │    基金经理    │ ◄── 最终决策
    └────────┬───────┘
             ▼
    ┌────────────────┐
    │     操盘手     │ ◄── 执行交易
    └────────────────┘
```

## 协作机制

1. **每日晨会** - 讨论当日交易机会
2. **周度复盘** - 评估上周策略表现
3. **月度策略会** - 制定下月投资计划
