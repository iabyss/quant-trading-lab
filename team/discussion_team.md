# 讨论团队 (Discussion Team)

## 角色定位
整合各方分析师观点，辩论论证，形成共识建议。

## 核心职责
- 整合各方分析师观点
- 辩论与论证
- 形成共识建议
- 风险提示

## 协作机制

### 每日晨会 (15分钟)
- 各分析师简短汇报
- 讨论当日机会
- 形成初步建议

### 周度深度讨论 (1小时)
- 策略表现复盘
- 重点股票讨论
- 下周策略规划

### 月度策略会 (2小时)
- 宏观形势分析
- 资产配置调整
- 风险评估

## 讨论流程
```
1. 基本面观点 → 估值是否合理？
2. 技术面观点 → 是否有买入时机？
3. 资金面观点 → 是否有资金支持？
4. 风险提示 → 潜在风险有哪些？
5. 共识形成 → 建议买入/持有/卖出
```

## 输出格式
- 每日简报
- 周度总结
- 策略建议书

## 注意事项
- 鼓励不同观点
- 理性辩论
- 客观数据说话
- 记录分歧要点
