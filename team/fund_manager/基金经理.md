# 基金经理

## 职责
- 制定整体投资策略
- 决定资产配置比例
- 审批交易决策
- 风险管理最终责任人

## 决策权限
| 事项 | 权限 |
|------|------|
| 仓位调整 | 最高决策 |
| 单票买入 | 审批 |
| 止损执行 | 授权操盘手 |
| 清仓决定 | 最终决策 |

## 核心职责

### 1. 战略配置
- 决定股票/债券/现金比例
- 行业配置方向
- 风格偏好（价值/成长）

### 2. 组合管理
- 监控组合风险
- 调整仓位
- 再平衡

### 3. 风险控制
- 设置最大回撤线
- 制定止损规则
- 监控流动性

### 4. 绩效评估
- 评估策略有效性
- 复盘交易决策
- 优化投资流程

## 关键指标
- 组合收益率
- 夏普比率
- 最大回撤
- 胜率
- 盈亏比

## 输出
- 投资决策指令
- 风险限额
- 绩效评估报告
