# 砖型图策略分析报告

## 策略公式解读

```
VAR1A := (HHV(HIGH,4) - CLOSE) / (HHV(HIGH,4) - LLV(LOW,4)) * 100 - 90
VAR2A := SMA(VAR1A, 4, 1) + 100
VAR3A := (CLOSE - LLV(LOW,4)) / (HHV(HIGH,4) - LLV(LOW,4)) * 100
VAR4A := SMA(VAR3A, 6, 1)
VAR5A := SMA(VAR4A, 6, 1) + 100
VAR6A := VAR5A - VAR2A
砖型图 := IF(VAR6A > 4, VAR6A - 4, 0)
```

### 核心逻辑

| 变量 | 含义 |
|------|------|
| HHV(HIGH,4) | 4日内最高价 |
| LLV(LOW,4) | 4日内最低价 |
| VAR1A | 当前价格相对4日区间的位置（顶部信号） |
| VAR3A | 当前价格相对4日区间的位置（底部信号） |
| VAR6A | 多空力量对比（正=多头，负=空头） |
| 砖型图 | 突破4以上的动能 |

### 交易信号

- **买入**: 砖型图从0开始增长（首次突破4）
- **卖出**: 砖型图从高点回落

## 策略分析

### ✅ 适用场景

1. **趋势明显的单边行情**
   - 上涨趋势中，砖型图持续增长 → 持仓不动
   - 下跌趋势中，砖型图持续下降 → 空仓观望

2. **波动率适中的市场**
   - 窄幅震荡时信号较少，趋势来临时信号准确
   - AAPL这类股票表现较好

3. **中长期趋势跟踪**
   - 配合周线/月线使用效果更好
   - 避免被短期波动洗掉

### ❌ 不适用场景

1. **高波动市场**
   - BTC、垃圾币等波动太大，频繁被洗出

2. **震荡行情**
   - 反复交叉，产生大量小额亏损

3. **短线频繁交易**
   - 信号过于灵敏，容易被庄家收割

## 优化建议

### 1. 增加过滤条件
```python
# 只在趋势方向交易
if MA20 > MA60:  # 上涨趋势
    只做多
if MA20 < MA60:  # 下跌趋势  
    只做空
```

### 2. 增加确认信号
```python
# 砖型图突破后，等待RSI确认
if 砖型图突破 AND RSI < 70:
    买入
```

### 3. 动态止盈止损
```python
# 跟踪止损
if 收益 > 10%:
    移动止损到成本价
```

### 4. 只做主流股票
- 避免小市值股票
- 避免加密货币

## 结论

| 维度 | 评分 | 说明 |
|------|------|------|
| 趋势跟踪 | ⭐⭐⭐ | 对趋势敏感，能抓住大行情 |
| 震荡适配 | ⭐ | 震荡行情表现差 |
| 风险控制 | ⭐⭐ | 需要配合止损使用 |
| 交易频率 | ⭐⭐⭐ | 频率适中 |

**总体评价**: 这是一个**趋势跟随型**指标，适合在单边行情中使用。建议配合均线过滤，只在趋势方向交易，并设置合理止损。
